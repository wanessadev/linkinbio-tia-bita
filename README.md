Uma página de **Link in Bio** totalmente personalizada, desenvolvida sob medida para a confeitaria artesanal **Doces da Tia Bita** (@docestiabita). O projeto foge dos templates genéricos do mercado, unindo uma abordagem moderna de UI/UX com a essência acolhedora e familiar da marca.

---

## 🎨 Identidade Visual & UI/UX

O design foi estruturado para gerar conexão emocional imediata e acessibilidade. 

* **Paleta de Cores:** Uso preciso de tons pastéis (bege e creme de fundo), marrom chocolate para os textos (garantindo excelente contraste e legibilidade) e rosa pastel vintage para os detalhes.
* **Hierarquia Estratégica:** Os botões foram organizados focando na jornada de conversão do cliente:
    1.  *Destaque Total:* Botão verde chamativo para pedidos diretos via WhatsApp.
    2.  *Quebra Visual Sutil:* Botão de Kits Festa com borda marrom chocolate para atrair o olhar para o produto de maior ticket médio.
    3.  *Clean & Minimalista:* Demais botões em rosa pastel e unificação das redes sociais em ícones no cabeçalho.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica da página.
* **CSS3:** Estilização moderna, uso de Flexbox, sombras suaves (*box-shadow*) e efeitos de transição (*hover*).
* **Font Awesome / SVGs:** Para a fileira minimalista de ícones das redes sociais.
